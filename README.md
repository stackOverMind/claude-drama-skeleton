# AI 内容创作多项目仓库

基于 Claude Code 的 AI 内容创作平台，支持小说、AI 漫剧、单集视频三种项目类型，覆盖从创意到成品的完整流水线。

## 项目类型

| 类型 | 说明 | 创作流程 |
|------|------|---------|
| **小说** | 分章节长篇小说，10-50 章，每章 3000-5000 字 | 大纲 → 正文 → 润色 |
| **AI 漫剧** | 多集连载动画短剧 | 小说 → 剧本 → 素材 → 关键帧 → 视频 |
| **单集视频** | 独立 AI 视频作品 | 脚本 → 素材 → 关键帧 → 视频 |

## 快速开始

### 前置条件

- [Claude Code](https://claude.ai/code)（CLI 或 VS Code 插件）
- 有效的 API Key

### 安装

```bash
git clone <repo-url>
cd claude-drama-skeleton
cp .env.example .env
# 编辑 .env，填入你的 API Key
```

### 创建项目

```
> 创建一个新项目
```

触发 `init-project`，选择项目类型并命名，自动生成标准化目录结构。

### 自然语言驱动创作

在 Claude Code 中直接用自然语言对话，Skill 自动匹配触发：

```
> 创作一篇都市异能小说
> 帮我写一部战神归来题材的微短剧，60集
> 分析这个剧本，提取所有角色和场景
> 生成女主角的定妆照，古风仙侠风格
> 把这张图改成赛博朋克风格
> 生成一段15秒的仙侠战斗视频
```

## 预装 Skills

项目在 `.claude/skills/` 下预装了 10 个 Skill，覆盖创作全流程：

### 项目初始化

| Skill | 触发方式 | 功能 |
|-------|---------|------|
| **init-project** | `> 创建项目` | 引导选择项目类型，生成标准化目录结构 |

### 内容创作

| Skill | 触发方式 | 对应阶段 | 功能 |
|-------|---------|---------|------|
| **chinese-novelist** | `/chinese-novelist` | 小说 | 分章节创作中文小说。递进式问答收集需求，自动生成大纲、人物档案，支持串行/并行写作，每章 3000-5000 字 |
| **short-drama** | `/short-drama` | 剧本 | 微短剧剧本创作。从选题到分集剧本，支持国内/出海双模式，13 种题材、付费卡点设计、节奏曲线、合规审核 |
| **seedance-storyboard-generator** | `/seedance-storyboard-generator` | 剧本+分镜 | 将小说/故事转换为 Seedance 2.0 视频脚本，生成标准剧本格式、资产生成提示词、分镜脚本 |

### 素材与视觉

| Skill | 触发方式 | 对应阶段 | 功能 |
|-------|---------|---------|------|
| **script-analyzer** | `/script-analyzer` | 剧本 → 素材 | 从剧本提取角色、场景、道具，生成定妆照描述和 AI 提示词，自动创建素材目录 |
| **content-gen** | `/content-gen` | 素材+视频 | AI 内容生成（图片+视频）。图片：Doubao Seedream，文生图/图生图；视频：Doubao Seedance，文生视频/图生视频/视频延长 |
| **seedance-prompt** | `/seedance-prompt` | 视频提示词 | 专业的 Seedance 2.0 视频提示词生成器，支持时间戳分镜、多模态引用、超长视频分段策略 |

### 音视频生成

| Skill | 触发方式 | 对应阶段 | 功能 |
|-------|---------|---------|------|
| **tts** | `/tts` | 音频 | AI 语音合成。基于小米 MiMo TTS，支持预置音色、音色设计、音色复刻，以及视频音频提取 |

## 典型工作流

### 小说项目

```
/init-project → 选择"小说"类型
    ↓
/chinese-novelist          # 递进式问答 → 大纲 → 逐章创作 → 1-小说/
```

### AI 漫剧项目

```
/init-project → 选择"AI 漫剧"类型
    ↓
/chinese-novelist          # 1. 创作小说 → 1-小说/
    ↓
/short-drama → /start      # 2. 选题定位
           → /plan        # 3. 生成创作方案
           → /episode 1   # 4. 逐集写剧本 → 2-剧本/
    ↓
/script-analyzer → /analyze all   # 5. 提取角色/场景/道具 → 3-素材/
    ↓
/content-gen                 # 6. 生成视觉素材图片 → 3-素材/
    ↓
/seedance-prompt           # 7. 生成视频提示词
/content-gen                 # 8. 生成视频 → 5-视频/
    ↓
/tts                       # 9. 配音合成 → 5-音频/
```

### 单集视频项目

```
/init-project → 选择"单集视频"类型
    ↓
/seedance-storyboard-generator   # 1. 生成脚本+分镜 → 1-脚本/
    ↓
/content-gen                       # 2. 生成素材图片 → 2-素材/
    ↓
/seedance-prompt                 # 3. 生成视频提示词
/content-gen                       # 4. 生成视频 → 4-视频/
    ↓
/tts                             # 5. 配音合成 → 5-音频/
```

## 项目目录结构

### 小说项目

```
项目-小说-{名称}/
├── 1-大纲/                    # 故事大纲与规划
│   ├── 大纲.md
│   └── 人物设定.md
├── 2-正文/                    # 分章节正文
│   ├── 第一章.md
│   └── ...
├── 3-资料/                    # 参考资料、世界观设定
├── 4-润色/                    # 修改稿、润色记录
└── 项目说明.md
```

### AI 漫剧项目

```
项目-漫剧-{名称}/
├── 1-小说/                    # 原文小说，按章节存放
├── 2-剧本/                    # 改编剧本
│   ├── .drama-state.json      # 创作状态追踪
│   ├── 创作方案.md
│   ├── 角色档案.md
│   ├── 分集目录.md
│   └── 第{N}集/
│       └── 剧本.md
├── 3-素材/                    # AI 生成素材
│   ├── 风格配置.json          # 画面风格配置
│   ├── 角色/{角色名}/
│   │   └── 描述.md            # 定妆照描述 + AI 提示词
│   ├── 场景/{场景名}.md
│   ├── 道具/{道具名}.md
│   └── 索引.json              # 素材注册表
├── 4-关键帧/                  # 按场景合成的关键帧（JSON）
│   └── 第{N}集/
│       └── 镜头{N}/
│           └── 帧{NNN}.json
└── 5-视频/                    # 最终视频输出
    └── 第{N}集/
        └── 镜头{N}.mp4
```

### 单集视频项目

```
项目-视频-{名称}/
├── 1-脚本/                    # 视频脚本/文案
│   └── 脚本.md
├── 2-素材/                    # AI 生成素材
│   ├── 角色/
│   ├── 场景/
│   └── 道具/
├── 3-关键帧/                  # 关键帧定义
├── 4-视频/                    # 最终视频输出
├── 5-音频/                    # 配音、音效、音乐
└── 项目说明.md
```

## 依赖

| 服务 | 用途 | 配置 |
|------|------|------|
| **Claude Code** | 驱动整个流水线的 AI 引擎 | `~/.claude/settings.json` |
| **Doubao Seedream** | AI 图片生成 | `.env` → `NEW_API_BASE_URL`, `API_KEY` |
| **Doubao Seedance** | AI 视频生成 | `.env` → `ARK_API_KEY` 或 `NEW_API_BASE_URL` |
| **Xiaomi MiMo TTS** | AI 语音合成 | `.env` → `MIMO_API_KEY` |

## 文件放置规则

- 所有内容文件必须放在某个 `项目-{类型}-{名称}/` 目录下
- 根目录仅允许存在：项目文件夹、`临时/`、`CLAUDE.md`、`README.md`、`.claude/`、配置文件
- 不确定归属的文件放入 `临时/`，后续整理归档
