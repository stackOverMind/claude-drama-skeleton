# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 提供项目指引。

## 项目简介

AI 内容创作多项目仓库。支持三种项目类型，每种类型有独立的目录结构和创作流程。

## 项目命名

- 前缀：`项目-`
- 格式：`项目-{类型}-{名称}/`
- 示例：`项目-视频-产品宣传片`、`项目-漫剧-古风剑客`、`项目-小说-都市异能`

## 项目类型与目录结构

### 类型一：单集视频

独立的单条 AI 视频作品。适用短视频、宣传片、创意视频。

```
项目-视频-{名称}/
├── 1-脚本/                    # 视频脚本/文案
│   └── 脚本.md
├── 2-素材/                    # AI 生成素材
│   ├── 角色/
│   ├── 场景/
│   └── 道具/
├── 3-关键帧/                  # 关键帧定义
├── 4-视频/                    # 最终视频输出
├── 5-音频/                    # 配音、音效、音乐
└── 项目说明.md
```

### 类型二：AI 漫剧

多集连载的动画短剧。流程：小说 → 剧本 → 素材 → 关键帧 → 视频。

```
项目-漫剧-{名称}/
├── 1-小说/                    # 原文小说，按章节存放
│   └── 第一章.md
├── 2-剧本/                    # 改编剧本
│   └── 第一集/
│       └── 镜头1.md           # 镜头拆分，含镜头列表
├── 3-素材/                    # AI 生成素材
│   ├── 角色/                  # 每个角色一个子目录
│   │   └── <角色名>/
│   │       ├── 描述.md         # 性格、背景、外貌文字稿
│   │       ├── 形象.jpg        # 角色形象图
│   │       └── 声音.wav        # 角色配音样本
│   ├── 场景/
│   └── 道具/
├── 4-关键帧/                  # 按场景合成的关键帧
│   └── 第一集/
│       └── 镜头1/
│           ├── 帧001.json      # 关键帧定义（布局、素材、变换）
│           └── 帧002.json
├── 5-视频/                    # 最终合成的视频
│   └── 第一集/
│       └── 镜头1.mp4
└── 项目说明.md
```

### 类型三：小说

分章节的长篇小说创作。支持 10-50 章，每章 3000-5000 字。

```
项目-小说-{名称}/
├── 1-大纲/                    # 故事大纲与规划
│   ├── 大纲.md
│   └── 人物设定.md
├── 2-正文/                    # 分章节正文
│   ├── 第一章.md
│   └── 第二章.md
├── 3-资料/                    # 参考资料、世界观设定
├── 4-润色/                    # 修改稿、润色记录
└── 项目说明.md
```

## 文件放置规则

- 所有内容文件必须放在某个 `项目-{类型}-{名称}/` 目录下
- 根目录仅允许存在：项目文件夹、`临时/`、`CLAUDE.md`、`README.md`、`.claude/`、配置文件
- 不确定归属的文件放入 `临时/`，后续再整理归档到对应项目
- 各阶段通过集/镜头目录关联：`1-小说/第一集.md` → `2-剧本/第一集/` → `4-关键帧/第一集/` → `5-视频/第一集/`
- 关键帧文件为 JSON，描述素材摆放、变换和时序，属于数据而非代码
- 剧本镜头内嵌 AI 图片生成提示词，保证可复现性
