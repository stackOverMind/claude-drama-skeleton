# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 提供项目指引。

## 项目简介

AI 漫剧制作流水线。将小说通过五个阶段转化为动画短剧：小说 → 剧本 → 素材 → 关键帧 → 视频。

## 流水线阶段

项目遵循线性内容流水线，每个阶段产出是下一阶段的输入：

1. **小说** (`1-小说/`) — 原文小说，按章节存放的 markdown 文件
2. **剧本** (`2-剧本/`) — 由小说章节改编的剧本。包含场景拆分、对白、镜头描述、AI 图片生成提示词
3. **素材** (`3-素材/`) — AI 生成的视觉素材：角色设计、场景、道具。每份素材附带元数据（生成提示词、参数、标签）
4. **关键帧** (`4-关键帧/`) — 按场景合成的关键帧：画面布局、角色位置、表情、镜头构图。对应剧本中的镜头
5. **视频** (`5-视频/`) — 最终合成的视频，按场景或整集输出

## 目录结构

```
1-小说/               # 第一阶段：小说原文
  第一章.md            # 按集/章节的 markdown

2-剧本/               # 第二阶段：改编剧本
  第一集/
    镜头1.md           # 镜头拆分，含镜头列表

3-素材/               # 第三阶段：AI 生成素材
  角色/               # 角色设定
    <角色名>/           # 每个角色一个文件夹
      描述.md           # 角色描述（性格、背景、外貌文字稿）
      形象.jpg          # 角色形象图
      声音.wav          # 角色配音样本
  场景/               # 场景背景图
  道具/               # 物件与道具图
  索引.json           # 素材注册表，记录元数据

4-关键帧/             # 第四阶段：关键帧合成
  第一集/
    镜头1/
      帧001.json      # 关键帧定义（布局、素材、变换）
      帧002.json

5-视频/               # 第五阶段：最终输出
  第一集/
    镜头1.mp4
```

## 命名约定

- 各阶段通过集/镜头目录关联：`1-小说/第一集.md` → `2-剧本/第一集/` → `4-关键帧/第一集/` → `5-视频/第一集/`
- 关键帧文件为 JSON，描述素材摆放、变换和时序，属于数据而非代码
- `3-素材/索引.json` 记录每份素材的来源提示词和生成参数
- 剧本镜头内嵌 AI 图片生成提示词，保证可复现性

## 文件放置规则

- 所有文件必须放在合适的目录下，不允许在项目根目录直接生成文件
- 如果不确定文件应归属哪个阶段目录，创建 `临时/` 目录放在其中，后续再整理归档
