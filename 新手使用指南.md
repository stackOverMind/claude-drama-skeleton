# AI 漫剧制作环境安装指南（面向初学者）

本指南帮助你从零开始搭建 AI 漫剧制作所需的开发环境。

---

## 第一步：安装 VS Code

VS Code 是免费的代码编辑器，我们用它来管理项目和与 Claude Code 交互。

### 下载

前往官方网站下载：https://code.visualstudio.com

### 各系统安装方法

| 系统 | 方法 |
|------|------|
| **macOS** | 下载 `.dmg` 文件，打开后将 Visual Studio Code 拖入「应用程序」文件夹 |
| **Windows** | 运行 `.exe` 安装程序，按提示完成安装 |
| **Linux (Debian/Ubuntu)** | 在终端运行 `sudo apt install code`，或从官网下载 `.deb` 安装包 |

安装完成后，打开 VS Code，确认能看到欢迎页面即可。

---

## 第二步：安装 Claude Code

Claude Code 是 Anthropic 官方的 AI 编程助手命令行工具。

### 系统要求

- macOS 13.0+ / Windows 10 1809+ / Ubuntu 20.04+
- 4 GB 以上内存

### 安装命令

打开终端（macOS 用「终端」，Windows 用 PowerShell），根据你的系统选择对应命令：

| 系统 | 命令 |
|------|------|
| **Windows / macOS / Linux** | `npm install -g @anthropic-ai/claude-code` |

> **Windows 安装注意事项**：
> 1. 先安装 [Node.js](https://nodejs.org/)（LTS 版本）和 [Git for Windows](https://git-scm.com/downloads/win)
> 2. 以**管理员身份**打开 PowerShell
> 3. 执行 `Set-ExecutionPolicy Unrestricted` 解除脚本执行限制
> 4. 再运行上面的 `npm install` 命令

### 验证安装

安装完成后，在终端（macOS 用「终端」，Windows 用 **PowerShell** 或 **Git Bash**）输入：

```bash
claude --version
```

如果显示版本号，说明安装成功。

## 第三步：配置 API 中转站

由于国内无法直接访问 Anthropic 官方 API，我们需要通过 **API 中转站** 来使用 Claude Code。

### 3.1 跳过 Claude Code 登录

使用中转站需要跳过 Claude Code 的官方登录流程。

创建或编辑文件 `~/.claude.json`：

| 系统 | 文件路径 |
|------|---------|
| **macOS / Linux** | `~/.claude.json` |
| **Windows** | `C:\Users\你的用户名\.claude.json` |

写入以下内容：

```json
{
  "hasCompletedOnboarding": true
}
```

### 3.2 配置 settings.json

创建或编辑 `~/.claude/settings.json`（如果 `.claude` 目录不存在，手动创建）：

| 系统 | 文件路径 |
|------|---------|
| **macOS / Linux** | `~/.claude/settings.json` |
| **Windows** | `C:\Users\你的用户名\.claude\settings.json` |

```json
{
  "env": {
    "ANTHROPIC_BASE_URL": "http://175.27.225.233/",
    "ANTHROPIC_AUTH_TOKEN": "你的API密钥",
    "ANTHROPIC_MODEL": "claude-opus-4-6",
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "claude-opus-4-6",
    "ANTHROPIC_DEFAULT_OPUS_MODEL": "claude-opus-4-6",
    "ANTHROPIC_DEFAULT_HAIKU_MODEL": "claude-opus-4-6"
  }
}
```

将 `你的API密钥` 替换为中转站提供的密钥。目前这个中转站没有正式公开运营，如果需要请联系作者。

> **注意**：如果之前设置了 `ANTHROPIC_AUTH_TOKEN`、`ANTHROPIC_BASE_URL` 等环境变量，请先清除：
> - **macOS / Linux**：从 `~/.zshrc` 或 `~/.bashrc` 中删除相关 `export` 语句
> - **Windows**：在「系统属性 → 高级 → 环境变量」中删除相关变量

### 3.3 验证配置

**关闭并重新打开终端**，然后运行：

```bash
claude
```

首次启动选择「Trust This Folder」。启动后输入 `/status` 查看配置，确认模型显示为 `claude-opus-4-6` 即可。


## 第四步：在 VS Code 中安装 Claude Code 插件

### 安装

有三种方式，任选其一：

1. **在 VS Code 内搜索安装**（最简单）
   - 打开 VS Code
   - 按 `Cmd+Shift+X`（macOS）或 `Ctrl+Shift+X`（Windows/Linux）打开插件面板
   - 搜索 `Claude Code`
   - 点击 **Install（安装）**

2. **命令行安装**
   ```bash
   code --install-extension anthropic.claude-code
   ```

3. **从网页安装**
   - 访问 https://marketplace.visualstudio.com/items?itemName=anthropic.claude-code
   - 点击页面上的 **Install** 按钮

> **要求**：VS Code 版本需 >= 1.98.0。

### 使用插件

安装完成后：

1. 在 VS Code 右上角工具栏找到 **claude 图标**，或在底部状态栏点击 **Claude Code**
2. 首次使用会提示登录，由于已配置 `~/.claude.json` 跳过登录，会直接进入对话界面
3. 如果提示需要登录，检查 `~/.claude.json` 文件是否正确创建

### 配置 VS Code 插件

打开 VS Code 设置，搜索 `Claude Code: Environment Variables`，然后在 `settings.json` 中手动配置（按 `Cmd+Shift+P` / `Ctrl+Shift+P`，输入 `Open User Settings (JSON)` 打开）：

```json
{
  "claudeCode.preferredLocation": "panel",
  "claudeCode.selectedModel": "claude-opus-4-6",
  "claudeCode.environmentVariables": [
    {
      "name": "ANTHROPIC_BASE_URL",
      "value": "http://175.27.225.233/"
    },
    {
      "name": "ANTHROPIC_AUTH_TOKEN",
      "value": "你的API密钥"
    },
    {
      "name": "ANTHROPIC_DEFAULT_SONNET_MODEL",
      "value": "claude-opus-4-6"
    },
    {
      "name": "ANTHROPIC_DEFAULT_OPUS_MODEL",
      "value": "claude-opus-4-6"
    },
    {
      "name": "ANTHROPIC_DEFAULT_HAIKU_MODEL",
      "value": "claude-opus-4-6"
    }
  ]
}
```

> 将 `你的API密钥` 替换为你的中转站 API Key。如果 CLI 的 `~/.claude/settings.json` 已正确配置，VS Code 插件会自动复用，也可以跳过此步骤。

## 第五步：克隆项目并开始创作

### 克隆项目

在终端中运行：

```bash
git clone https://github.com/stackOverMind/claude-drama-skeleton.git
cd claude-drama-skeleton
```

> **Windows 用户**：确保已安装 [Git for Windows](https://git-scm.com/downloads/win)，然后在 PowerShell 或 Git Bash 中运行以上命令。也可以直接在 VS Code 的内置终端中运行。

### 在 VS Code 中打开项目

在 VS Code 中，点击菜单 **File → Open Folder**（或使用快捷键 `Cmd+O` / `Ctrl+O`），选择刚才克隆下来的 `claude-drama-skeleton` 文件夹。

### 配置项目 Skill 环境变量

项目中自定义的 Skill 需要读取 `.env` 文件中的密钥才能正常运行。请在项目根目录下执行：

```bash
cp .env.example .env
```

然后用编辑器打开 `.env` 文件，将其中的占位符替换为你的真实 API Key。

> **注意**：`.env` 文件包含敏感密钥，已在 `.gitignore` 中排除，不会被提交到 Git 仓库。

### 开始使用 Claude Code 创作

打开项目后，点击 VS Code 中的 Claude Code 图标，即可开始与 AI 对话。你可以：

1. 让 Claude 根据小说章节生成剧本（`1-小说/` → `2-剧本/`）
2. 让 Claude 生成 AI 图片提示词和素材描述（`2-剧本/` → `3-素材/`）
3. 让 Claude 合成关键帧定义（`3-素材/` → `4-关键帧/`）
4. 让 Claude 生成最终视频合成方案（`4-关键帧/` → `5-视频/`）

项目结构和流水线说明详见 [CLAUDE.md](CLAUDE.md)。

- [ ] 项目已克隆并在 VS Code 中打开
