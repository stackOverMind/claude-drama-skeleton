# Skill 使用指南

通过实际示例展示每个 Skill 的用法。在 Claude Code 中直接输入自然语言或斜杠命令即可触发。

---

## 1. 创建项目 — init-project

```
> 创建一个漫剧项目，叫古风剑客
```

自动生成标准目录结构：

```
项目-漫剧-古风剑客/
├── 1-小说/
├── 2-剧本/
├── 3-素材/{角色/, 场景/, 道具/}
├── 4-关键帧/
├── 5-视频/
└── 项目说明.md
```

其他示例：

```
> 新建一个小说项目，叫都市异能
> 帮我创建一个产品宣传片单集视频项目
```

---

## 2. 小说创作 — chinese-novelist

```
> 帮我写一部悬疑小说，30章，每章4000字
```

流程：递进式问答收集需求 → 自动生成大纲、人物档案 → 逐章创作。支持断点续写：

```
> 继续之前没写完的小说
```

输出到 `1-小说/` 目录。

---

## 3. 微短剧剧本 — short-drama

斜杠命令驱动，按阶段推进：

```
/short-drama

/start              # 选题定位：选题材、受众、集数
/plan               # 生成创作方案、三幕结构
/characters         # 生成角色档案 + 人物关系图
/outline            # 逐集大纲，标🔥高能集和💰付费卡点
/episode 1          # 写第1集剧本
/episode 5-8        # 批量写第5-8集
/review 5           # 评分第5集（5维度50分制）
/compliance         # 合规审核
/export             # 导出完整剧本
```

---

## 4. 剧本分析 — script-analyzer

从已完成的剧本中提取素材信息，生成 AI 绘图提示词。

```
> 分析这个剧本

/analyze style            # 先选视觉风格（必选）
/analyze characters       # 提取所有角色，生成定妆照描述
/analyze scenes           # 提取场景
/analyze props            # 提取道具
/analyze all              # 一键全部提取
```

支持 8 种风格：写实、动漫、条漫、水墨、3D卡通、3D写实、赛博朋克、扁平插画。

输出到 `3-素材/`：

```
3-素材/
├── 风格配置.json
├── 角色/林小染/描述.md    # 含正面/负面 AI 提示词
├── 场景/古镇酒肆.md
├── 道具/青锋剑.md
└── 索引.json
```

## 5. 图片生成 — content-gen

触发时会弹出对话框询问 provider 和 model。

### 文生图

```
> 生成一张赛博朋克风格的城市夜景图，保存到 项目-漫剧-x/3-素材/场景/赛博城市.png
```

### 图生图（只写改动点）

```
> 基于 3-素材/角色/主角/形象.png，改成冲锋姿势，保存到 3-素材/角色/主角/冲锋.png
```

### 常用 provider 和 model

| Provider | Model | 说明 |
|----------|-------|------|
| cool | gpt_image_2 | GPT 图片模型 |
| cool | seedream_4_5 | 字节跳动 Seedream |
| cool | midjourney_v7 | Midjourney |
| newapi | doubao-seedream-5-0-260128 | 豆包 Seedream |

## 6. 视频生成 — content-gen

同样触发 provider/model 对话框。工作流：写 prompt 文件 → 用户确认 → 提交 → 自动轮询下载。

### 图生视频

```
> 用 3-素材/角色/主角/形象.png 生成一段5秒的视频，主角转身挥剑
```

### 文生视频

```
> 生成一段10秒的仙侠战斗视频，两人在悬崖边对决
```

### 视频到视频（需先上传参考视频到 OSS）

```
> 参考这个视频，把狗头换成猫头
```

### 常用 provider 和 model

| Provider | Model | 说明 |
|----------|-------|------|
| cool | seedance_2 | Seedance 2.0（高质量） |
| cool | seedance_2_fast | Seedance 2.0 快速版 |
| cool | kling_3_sora | 可灵 3.0 |
| cool | veo_3_1_fast | Google Veo 3.1 |
| volcengine | doubao-seedance-2-0-260128 | 火山引擎版 Seedance |
| newapi | doubao-seedance-2-0-260128 | new-api OpenAI 兼容接口 |

## 7. 视频提示词 — seedance-prompt

为 Seedance 2.0 生成专业视频提示词，返回 2-3 个变体。

```
> 帮我写一段15秒仙侠战斗视频的提示词
> 做一个30秒奶茶产品广告，需要图片首帧
> 把这个视频延长10秒，加一段男主反杀剧情
```

关键技巧：
- 单次上限 15 秒，超长视频用分段延长（`将@视频1延长15s`）
- 9 秒以上推荐时间轴分镜：`0-3秒: ... 4-8秒: ...`
- 用 `@图片1`、`@视频1`、`@音频1` 引用素材

## 8. 分镜脚本生成 — seedance-storyboard-generator

将小说/故事转换为完整的视频脚本 + 分镜提示词。

```
> 把这篇林冲的故事改成5集AI漫剧分镜
> 我有一个30秒短剧大纲，帮我生成完整剧本和分镜
> 为我的小说生成20集的Seedance分镜脚本
```

输出：
- `剧本.md` — 标准剧本格式（`△` 镜头标记）
- `素材清单.md` — 角色/场景/道具 AI 提示词（C01-C99, S01-S99, P01-P99）
- `E01_分镜.md` — 每集上传清单 + 时间轴 Seedance 提示词

## 9. 语音合成 — tts

基于小米 MiMo TTS，支持三种模式。

### 预置音色

```
> 用冰糖的声音说"你好，欢迎来到午夜电台"，保存到 5-音频/开场白.wav
```

### 音色设计（用文字描述声音）

```
> 生成一段配音，声音是"三十多岁男性，低沉磁性，深夜DJ"，内容是"欢迎来到午夜电台"
```

### 音色复刻

```
> 用 5-音频/样本.mp3 的声音，复刻说"第二幕开场白"
```

### 从视频提取音频

```
> 把 5-视频/第一集/镜头1.mp4 的音频提取出来，从1分30秒开始，取10秒
```

### 内置情感标签

在文本中直接用括号标注情感：

```
> 用茉莉的声音说"(开心)今天天气真好！（深呼吸）不过...（悲伤）我有点想家了"
```

## 完整工作流示例

### 从零创建一部 10 集仙侠漫剧

```
# 1. 创建项目
> 创建一个漫剧项目，叫仙侠剑仙

# 2. 写小说
> 写一部仙侠小说，10章，主角是废柴逆袭的剑修

# 3. 编剧本
/short-drama
/start              # 选仙侠题材，10集
/plan               # 生成方案
/episode 1-10       # 逐集写剧本

# 4. 提取素材
/script-analyzer
/analyze style      # 选水墨风格
/analyze all        # 一键提取角色/场景/道具

# 5. 生成图片素材
> 生成所有角色的定妆照

# 6. 生成视频
> 为第1集的镜头1生成5秒视频

# 7. 配音
> 为第1集的镜头1生成配音
```
