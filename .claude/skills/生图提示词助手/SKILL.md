# AI生图助手

此技能通过阅读和应用工具特定文档来帮助生成专业的AI图像提示词。

## 何时使用

当用户出现以下情况时使用此技能：
- 想要使用特定的AI工具生成图片（即梦、通义万相、可灵、Google、DALL-E、Stable Diffusion等）
- 需要为特定图像生成平台格式的提示词
- 询问具有工具特定要求的提示词生成帮助

## 如何使用

### 步骤1：识别工具

从用户请求中提取图像生成工具名称。常见工具包括：
- 即梦
- 通义万相
- 可灵
- Google
- Stable Diffusion

### 步骤2：读取工具文档

在工作区查找对应的文档文件：
- 路径：`references/`
- 文件名：`[工具名称]官方提示词文档.md`
- 示例：`即梦官方提示词文档.md`、`通义万相AI作画—创作指南（含提示词示例）.md`、`可灵AI使用指南-图片生成.md`、`[Google]官方提示词文档.md`

使用 `Read` 工具读取文档。

### 步骤3：理解工具要求

阅读文档后，识别：
- 偏好的提示词结构（自然语言、关键词、公式）
- 特殊格式要求
- 推荐的词汇和术语
- 文档中提到的最佳实践和技巧
- 任何特定的公式或模板

### 步骤4：生成提示词

基于文档和用户的描述：

1. **应用工具格式**：遵循文档中推荐的结构
2. **使用推荐术语**：融入文档中的词汇
3. **遵循最佳实践**：应用文档中提到的技巧
4. **保持用户意图**：确保提示词捕捉用户的期望结果



### 步骤5：提供上下文

交付生成的提示词时，解释：
- 引用了哪个文档
- 应用了哪些具体指南
- 为什么提示词这样构建
- 任何权衡或考虑因素

## 参考资料

有关详细的工具特定指南，请参阅：
- `references/prompt-generation-guide.md` - 全面的提示词生成策略

## 错误处理

### 文档未找到

如果找不到工具文档：
```
抱歉，我找不到[工具名称]的官方提示词文档。
请验证：
1. 文档文件存在于工作区目录
2. 文件命名为：[工具名称]官方提示词文档.md

如果您有不同的文件名，请提供确切的名称。
```

### 读取失败

如果无法读取文档：
```
抱歉，我无法读取[工具名称]的提示词文档。
错误：[具体错误消息]

请检查文件是否损坏或被其他程序使用。
```

## 最佳实践

1. **始终先读取文档** - 绝不跳过文档读取步骤
2. **严格遵循文档** - 应用工具的特定要求
3. **明确来源** - 引用文档中指导提示词的哪些部分
4. **提供替代方案** - 如适用，提供多个提示词版本
5. **解释选择** - 帮助用户理解为什么提示词这样构建

## 使用示例

**用户请求：**
"请使用即梦生成：一位身着古装的女孩在樱花树下弹古琴，具有古风氛围和柔和光线"

**助手操作：**
1. 识别工具：即梦
2. 读取：`即梦官方提示词文档.md`
3. 理解：即梦偏好自然语言 + 简短美学关键词
4. 按照文档生成提示词
5. 提供解释并交付
