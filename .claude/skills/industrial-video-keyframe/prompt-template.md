# 关键帧生成提示词模板

```markdown
# 关键帧-{镜头名}-{版本} Prompt

## 参考图

- **图片1**：{角色名} 人物参考（{特征描述}）
- **图片2**：{场景名} 场景参考（{视角/时间}）

## 视觉风格

{项目风格提示词}

## 人物外貌

{从角色参考图提取的外貌描述}

## 环境描述

{从场景参考图提取的环境描述}

## 人物动作

{具体动作描述}

## 光影构图

{镜头角度、景别、光源、构图}

## 负面 Prompt

低质量，变形，多余肢体，手指畸形，模糊，水印，文字，签名，灰暗色调，过度曝光，阴影过硬，面部崩坏，五官不对称，比例失调，背景杂乱，噪点，油画质感，西方卡通风格，3D游戏塑料感，渲染过度
```
