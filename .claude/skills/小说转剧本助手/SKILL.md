# AI小说改剧本助手

此技能通过阅读和应用类型特定的改编指南，帮助将小说文本转换为专业剧本。

## 何时使用

当用户出现以下情况时使用此技能：
- 想要将小说文本转换为特定的剧本格式（漫剧、影视、动画、舞台剧、广播剧等）
- 需要特定媒体类型的剧本改编指导
- 询问帮助将叙述性散文转换为基于对话的剧本格式

## 如何使用

### 步骤1：识别剧本类型

从用户请求中提取目标剧本类型。常见类型包括：
- 漫剧
- 影视
- 动画
- 舞台剧
- 广播剧

### 步骤2：读取改编文档

在工作区查找对应的文档文件：
- 路径：`references/`
- 文件名：`小说如何改为漫剧剧本.md`

使用 `Read` 工具读取文档。

### 步骤3：理解剧本要求

阅读文档后，识别：
- 所需的剧本格式和结构
- 场景标题规范
- 对话格式规则
- 动作/叙述描述风格
- 该剧本类型的任何特定指南

### 步骤4：分析小说内容

解析小说文本以提取：
- 人物及其关系
- 情节发展和关键场景
- 对话台词
- 场景描述
- 情感节拍和氛围

### 步骤5：生成剧本

基于文档和小说分析：

1. **遵循格式规则**：应用目标剧本类型的特定结构
2. **保持核心情节**：保留基本的叙述元素
3. **调整表达风格**：将文学散文转换为视觉/对话语言
4. **增强视觉元素**：添加适合该媒介的动作和视觉描述

常见剧本格式：
- **漫剧**：场景、人物、镜头、对白、音效、画外音
- **影视**：场景标题（Scene #）、人物、动作描述、对话、括号说明
- **动画**：类似于影视，但可能包含更多奇幻元素
- **舞台剧**：幕、场景、人物、舞台指示、对话
- **广播剧**：集、人物、音效、对话、旁白

### 步骤6：提供说明

交付剧本时，解释：
- 进行了哪些转换以及原因
- 文档的哪些部分指导了改编
- 做出的任何创意决定
- 进一步完善的建议

## 错误处理

### 文档未找到

如果找不到剧本改编文档：
```
抱歉，我找不到[剧本类型]的改编指南。
请验证：
1. 文档文件存在于工作区目录
2. 文件命名为：小说如何改为[剧本类型]剧本.md

如果您有不同的文件名，请提供确切的名称。

如果此剧本类型没有改编文档，我可以：
1. 使用通用剧本改编原则
2. 帮助您创建改编文档模板
```

### 读取失败

如果无法读取文档：
```
抱歉，我无法读取[剧本类型]的改编指南。
错误：[具体错误消息]

请检查文件是否损坏或被其他程序使用。

如果问题持续存在，我可以尝试使用通用剧本改编原则。
```

### 小说内容不足

如果提供的小说内容过于简短：
```
您提供的小说内容相当简短。我建议：
1. 提供更完整的内容（至少一个完整的场景）
2. 指定需要改编的场景范围
3. 说明目标时长或集数

请提供更多信息，以便我可以生成更完整的剧本。
```

## 改编原则

1. **视觉转换** - 将抽象描述转换为具体的视觉元素
2. **对话保留** - 保留重要对话，必要时调整语调
3. **节奏控制** - 为目标媒介调整叙述节奏
4. **情感传达** - 使用视觉/对话语言表达情感
5. **尊重原作** - 保留创作者的意图和核心情节

## 改编技巧

### 可视化抽象概念
- "感到悲伤" → "眼眶含泪、低头不语"
- "思考了很久" → "来回踱步、眉头紧锁"
- "气氛紧张" → "镜头特写、背景音效压抑"

### 场景拆分
- 将复杂的小说场景分解为多个镜头
- 根据目标类型要求调整场景顺序
- 合理安排转场和剪辑

### 人物塑造
- 通过动作和对话展现个性
- 使用镜头角度和镜头尺寸增强人物存在感
- 保持一致的人物行为和逻辑

## 质量保证

交付剧本之前：
- [ ] 已阅读改编文档
- [ ] 剧本遵循目标类型的格式要求
- [ ] 核心情节和人物关系得到保留
- [ ] 视觉/对话语言适合该媒介
- [ ] 提供了改编说明

**用户请求：**

'这个世界，真的会有奇迹吗？'
雨声淅淅沥沥，仿佛在回答她的问题。"

**助手操作：**
1. 识别剧本类型：漫剧
2. 读取：`小说如何改为漫剧剧本.md`
3. 理解格式：场景标题、人物、镜头、对白、音效
4. 生成带有详细视觉描述的剧本
5. 交付并说明所做的转换
