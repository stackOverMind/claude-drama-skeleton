# Init Project: 项目初始化助手

## 功能

在当前仓库下创建标准化的子项目目录结构，为后续创作做好准备。

## 项目命名规则

- 项目目录前缀：`项目-`
- 完整格式：`项目-{名称}/`
- 示例：`项目-日漫女孩`、`项目-古风剑客`、`项目-都市异能`

## 使用流程

### 第一步：收集项目信息

**必须依次询问以下内容，不可跳过**：

1. **项目名称**
   ```
   请为项目命名（例如：日漫女孩、古风剑客、产品宣传片）：
   ```

2. **画面比例**
   ```
   请选择画面比例：
   - 9:16（竖屏，适合短视频/微短剧）
   - 16:9（横屏，适合常规视频/宣传片）
   ```

3. **视觉风格**
   ```
   请描述项目的视觉风格（例如：日漫二次元、古风仙侠、赛博朋克、真人写实、迪士尼3D动画等）：
   ```

4. **世界观设定**（可选，但建议填写）
   ```
   请描述项目的世界观设定（时代背景、地域、核心规则等），如暂无可先跳过，后续在「项目说明.md」中补充：
   ```

### 第二步：创建项目目录

根据用户输入，在项目根目录下创建对应的项目目录结构，并将收集到的信息填入 `项目说明.md`。

---

## 项目目录结构

```
项目-{名称}/
├── 1-脚本/                    # 视频脚本/文案/剧本
├── 2-素材/                    # AI 生成素材（角色、场景、道具等）
├── 3-故事板/                  # 故事板/关键帧定义
├── 4-视频/                    # 最终视频输出
└── 项目说明.md                # 项目基本信息与规则
```

---

## 项目说明.md 模板

创建项目时，同时生成 `项目说明.md`，将用户提供的视觉风格、画面比例、世界观设定直接填入：

```markdown
# {项目名称}

## 项目类型
[在此填写项目类型，如：单集视频 / AI 漫剧 / 小说]

## 项目描述
[在此填写项目简介]

## 画面比例
{用户选择的画面比例：9:16 或 16:9}

## 画面风格提示词
{用户提供的视觉风格描述}

## 世界观设定
{用户提供的世界观设定，若未提供则显示「[待补充]」}

## 备注
- 以上信息（画面比例、视觉风格、世界观）可在本文件中随时修改
- 修改后，后续 AI 生成素材将以最新内容为准
```

---

## 执行指令

1. **收集信息**：按顺序询问项目名称、画面比例、视觉风格、世界观设定
2. **检查冲突**：确认当前目录下是否已存在同名项目，如存在则提示修改
3. **创建目录**：按模板创建完整的目录结构
4. **生成文件**：创建 `项目说明.md`，将用户输入的内容直接填入对应字段
5. **确认完成**：向用户报告创建结果，列出目录结构，并提醒：
   > 项目已创建完成！画面比例、视觉风格、世界观等信息已写入 `项目说明.md`，后续如需修改可直接编辑该文件。

## 注意事项

- 所有项目目录以 `项目-` 为前缀，便于识别和管理
- 子目录均为空目录，由用户在后续创作中填充内容
- **创建时必须与用户确认**：项目名称、画面比例、视觉风格、世界观设定
- 创建完成后，明确告知用户以上信息可在 `项目说明.md` 中随时修改
