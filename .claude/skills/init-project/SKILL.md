# Init Project: 项目初始化助手

## 功能

引导用户选择项目类型，创建标准化的项目目录结构，为后续创作做好准备。

## 支持的项目类型

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| **单集视频** | 独立的单条 AI 视频作品 | 短视频、宣传片、单条创意视频 |
| **AI 漫剧** | 多集连载的动画短剧 | 系列短剧、连载动画、IP 改编 |
| **小说** | 分章节的长篇小说创作 | 网络小说、长篇小说、连载故事 |

## 项目命名规则

- 项目目录前缀：`项目-`
- 完整格式：`项目-{类型标识}-{自定义名称}/`
- 示例：`项目-视频-产品宣传片`、`项目-漫剧-古风剑客`、`项目-小说-都市异能`

## 使用流程

### 第一步：选择项目类型

向用户展示三种项目类型，请用户选择：

```
请选择要创建的项目类型：

1. 【单集视频】—— 独立的单条 AI 视频作品
   适用：短视频、宣传片、创意视频

2. 【AI 漫剧】—— 多集连载的动画短剧
   适用：系列短剧、连载动画、IP 改编
   流程：小说 → 剧本 → 素材 → 关键帧 → 视频

3. 【小说】—— 分章节的长篇小说创作
   适用：网络小说、长篇小说、连载故事
   支持：10-50章，每章3000-5000字

请输入数字 1/2/3 选择项目类型：
```

### 第二步：确认项目名称

用户选择类型后，询问自定义项目名称：

```
请为项目命名（例如：古风剑客、都市异能、产品宣传片）：
```

### 第三步：创建项目目录

根据用户输入，创建对应的项目目录结构。

## 项目目录结构模板

### 类型一：单集视频

```
项目-视频-{名称}/
├── 1-脚本/                    # 视频脚本/文案
│   └── 脚本.md
├── 2-素材/                    # AI 生成素材
│   ├── 角色/
│   │   └── .gitkeep
│   ├── 场景/
│   │   └── .gitkeep
│   ├── 道具/
│   │   └── .gitkeep
├── 3-关键帧/                  # 关键帧定义
│   └── .gitkeep
├── 4-视频/                    # 最终视频输出
│   └── .gitkeep
├── 5-音频/                    # 配音、音效、音乐
│   └── .gitkeep
└── 项目说明.md                # 项目基本信息
```

│   └── .gitkeep
├── 3-素材/                    # AI 生成素材
│   ├── 角色/
│   │   └── .gitkeep
│   ├── 场景/
│   │   └── .gitkeep

### 类型三：小说

```

├── 1-大纲/                    # 故事大纲与规划
│   ├── 大纲.md
│   └── 人物设定.md

│   └── .gitkeep

│   └── .gitkeep
├── 4-润色/                    # 修改稿、润色记录
│   └── .gitkeep
└── 项目说明.md                # 项目基本信息
```

## 项目说明.md 模板

创建项目时，同时生成 `项目说明.md`：

```markdown
# {项目名称}

## 项目类型
{单集视频 / AI 漫剧 / 小说}

## 创建日期
{YYYY-MM-DD}

## 项目描述
[在此填写项目简介]

## 创作状态
- [ ] 进行中
- [ ] 已完成
- [ ] 已暂停

## 备注
[其他需要记录的信息]
```

## 执行指令

1. **展示选项**：向用户展示三种项目类型，等待选择
2. **收集名称**：询问用户自定义项目名称
3. **检查冲突**：确认当前目录下是否已存在同名项目，如存在则提示修改
4. **创建目录**：按模板创建完整的目录结构
5. **生成文件**：创建 `项目说明.md`
6. **确认完成**：向用户报告创建结果，列出目录结构

## 注意事项

- 所有项目目录以 `项目-` 为前缀，便于识别和管理
- 使用 `.gitkeep` 占位空目录，确保目录结构可被 git 追踪
- 创建完成后，提示用户可以使用对应的 Skill 开始创作：
  - 小说项目 → `/chinese-novelist`
  - 漫剧项目 → `/short-drama`（剧本）、`/script-analyzer`（分析）、`/content-gen`（素材+视频）
  - 单集视频 → `/content-gen`、`/tts`
