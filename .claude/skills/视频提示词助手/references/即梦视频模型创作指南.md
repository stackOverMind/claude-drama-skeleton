# 即梦视频模型创作指南

本文讨论了即梦视频 3.0 Pro 模型的使用指南，涵盖了提示词写作、动作控制、情绪演绎、场景描述、风格设定、镜头语言、景别视角、美感控制、切镜能力和创意特效等方面。

## 关键要点

1. **提示词写作**：文生视频提示词为主体 + 动作 + 场景 + 镜头 + 风格 + （情绪演绎）+ （照明）；图生视频为动作 + 镜头 + （情绪/照明），越明确细节越多画面还原度越高。

2. **动作控制**：分为单一动作和时序组合动作，单一动作可添加运动相关提示词增强表现力；时序组合动作需按动作时序描述，输入超三个动作模型响应可能下降。

3. **情绪演绎**：模型能精准呈现微妙表情变化，但对部分情绪词不敏感；可通过基础提示词编写和进阶玩法构建情绪饱满剧情。

4. **场景描述**：T2V 可自由构建场景但需精确描述，I+T2V 画面美感高但受初始图片限制；场景构成要素包括地点、时间和氛围。

5. **风格设定**：T2V 能直出多种风格，I+T2V 可延续图片美学风格，模型对风格核心特征识别复现能力强，智能补全功能出色。

6. **镜头语言**：基础运镜能执行推、拉等指令，但处理特殊机位和复杂人物动作能力不足；进阶运镜可组合多个镜头，增强视觉节奏。

7. **创意特效**：通过首尾帧设计和 I+T2V 模态扩展打造特效，首尾帧有基础类型和创意模块，I+T2V 可让静态画面有动态效果。

---

## 即梦视频模型概述

✨ **即梦视频 3.0 Pro** 具有强大的语意遵循能力，写好prompt可以生成丰富多样的视频。

即梦视频Prompt写作的核心是"直观表达出你想要的效果"，可以使用自然语言自由书写出动作/运镜/风格/视角，从而有效的控制视频效果。如果遇见响应不好的情况，可尝试将prompt改得更加直白易懂。

---

## 提示词构建总结

### 文生视频
```
主体 + 动作 + 场景 + 镜头 + 风格 + （情绪演绎）+ （照明）
```

### 图生视频
```
动作 + 镜头 + （情绪/照明）
```

### 写好Prompt的核心思路

- 对于文生视频来说，提示词写得越明确、细节越多，画面还原度就越高。你可以指定主体、动作、运镜、光照、情绪等各类信息，模型会尽力还原你脑海中的构图、节奏与风格。

- 即使没有特别具体的画面构想，也可以给出一个简单的场景描述，让模型自动补充内容，快速探索创意方向。

- 而图生视频则更多依赖图像本身的信息，提示词主要用来引导人物动作、镜头语言或情绪变化。虽然风格大致由原图决定，但你仍然可以控制角色移动、镜头推进或光照变化等动态效果，实现静图到动态效果的自然过渡。

## 如何控制单视频生成

### 1. 动作指令

本指南将动作划分为**单一动作**、**时序组合动作**两种类型，且这两种动作类型涵盖简单和复杂两种不同难度。下方推荐了prompt的起手书写方式，关键在于简洁明了地描述动作，避免因prompt冗长或动作指令过于复杂而导致视频呈现的动作效果欠佳。

#### 1.1 单一动作（简单动作）

推荐I+T2V基础起手式：`"主体"+"单动作"`

推荐T2V基础起手式：`"清晰的主体描述"+"场景"+"单动作"`

进阶："动作"指令可添加如"快速""平稳地""用力地""向前"等运动速度、运动方向和运动幅度方面的提示词，以增强动作表现力。

**图+文生视频（I+T2V）：**
- ✅ 能够依据prompt和图片信息精确响应生成不同的动作类型，主体动作效果生动且准确。
- ❌ 对于高难度动作，需要谨慎对待！如跑酷、跳舞、打斗等动态幅度较大或需要交互的动作，仍存在打击感不足、肢体结构破坏严重等不稳定性。

**文生视频（T2V）：**
上下限均较高，在能够输出逼真生动动作的同时，也容易出现动作指令遵循不准确和动作幅度不可控等问题。

#### 1.2 时序组合动作（复杂动作）

按照动作的先后时序进行描述。

#### 2. 情绪演绎

模型能够精准呈现微妙表情变化，但对部分情绪词不敏感。建议：

- 通过基础提示词编写和进阶玩法构建情绪饱满的剧情。
- 使用具体的动作描述来间接表达情绪。
- 场景氛围的营造也能辅助情绪的表达。

#### 3. 场景描述

- **T2V**：可自由构建场景，但需要精确描述。场景构成要素包括地点、时间和氛围。
- **I+T2V**：画面美感高，但受初始图片限制。

#### 4. 风格描述

##### 4.1 风格直出（T2V）

能够直接生成多种风格，包括但不限于：
- 写实风格
- 漫画风格（美漫、日漫、韩漫）
- 3D动画风格（Low Poly、CG渲染）
- 艺术风格（油画、水彩、毛毡）
- 特殊风格（像素、体素、故障艺术、中式梦核）

##### 4.2 风格延续（I+T2V / I2V）

- 能够延续图片的美学风格
- 模型对风格核心特征识别复现能力强
- 智能补全功能出色

#### 5. 镜头语言

##### 5.1 基础运镜

- 推镜头：镜头向前移动
- 拉镜头：镜头向后移动
- 摇镜头：镜头水平移动
- 甩镜头：快速摇摄

##### 5.2 进阶运镜

- 环绕拍摄：围绕主体进行360度环绕
- 焦点转移：镜头焦点在不同物体间切换
- 变焦推进：镜头快速变焦向前推进
- 组合镜头：多个镜头组合使用

##### 5.3 景别和视角控制

- 航拍视角
- 俯拍视角
- 低角度仰拍
- 第一人称视角
- 微距镜头
- 水下视角
- FPV视角

#### 6. Prompt控制美感

##### 6.1 人物外形

可以通过Prompt精确控制人物的外形特征，包括但不限于：
- 年龄、性别
- 服饰风格
- 身体特征
- 表情神态
- 动作姿态

示例：
- 写实，穿着洛丽塔裙的甜美少女
- 戴虎头帽、穿着传统肚兜的可爱萌娃
- 写实，一位戴着金丝眼镜、身着白大褂的温柔女医生

##### 6.2 画面美感

- 光影效果：自然光、人造光、氛围光
- 色彩运用：冷暖色调、饱和度、对比度
- 构图方式：对称、三分法、引导线
- 材质表现：光滑、粗糙、透明、发光
- 纹理细节：清晰度、颗粒感

#### 7. 切镜能力

- **切镜能力**：模型支持在视频中切换不同的场景或画面
- **一镜到底**：连续的镜头不间断
- **场景转换**：平滑的场景过渡效果

#### 8. 创意特效（【即梦AI】特效玩法合集）

##### 8.1 首尾帧类型模块（基础分类）

通过设计首尾帧来实现创意转换：

- 物体变形：如狗狗变成3D拍立得玩具
- 性别转换：男人微笑着面部缓缓变成女性形象
- 逆生长过程
- 人物出现/消失：左边缓缓走出来一名美女
- 环境突变：富士山发生大爆炸
- 角色变形：小猫在金色烟雾中站立变形，猫耳渐变鸭舌帽

##### 8.2 首尾帧创意玩法模块（玩法应用）

- 玩具包装效果
- 人物拟人化：小狗狗拟人化的穿着保安制服
- 物体液态化：水果像岩浆般开始沸腾
- 挤压变形：像被挤压的捏捏乐般出现凹陷
- 杂志封面人物走出：镜头从特写杂志封面开始，缓慢拉远
- 建筑包裹：卢浮宫被巨型冰晶包裹
- 像素化转换：人物轮廓拆解成彩色方块
- 身体变异：背后生长出火焰翅膀
- 魔法特效：粉色海棠妙笔生花

##### 8.3 I+T2V 创意特效模块（扩展）

- 让静态画面产生动态效果
- 智能补全动作细节
- 增强画面表现力

## 实用技巧

1. **从简单开始**：先尝试单一动作和简单场景，逐步增加复杂度
2. **明确描述**：使用具体、清晰的描述，避免模糊的表达
3. **合理控制**：动作指令不要过于复杂，建议控制在3个以内
4. **多次尝试**：同一个prompt可以尝试生成多次，选择最佳效果
5. **组合使用**：灵活组合不同模块的功能，创造独特的视觉效果
