# 关键帧生成规则

## 命名规范

- 关键帧图：`关键帧-{镜头名}-{版本}.png`
- 提示词文件：`关键帧-{镜头名}-{版本}.md`
- 存放目录：`<项目名>/3-关键帧/`

## 参考图规则

- 基于角色生成新姿势 → 必须传入角色参考图
- 基于场景生成角色动作 → 必须传入场景参考图
- 角色在场景中做动作 → 角色参考图 + 场景参考图都必须传入
- 任何涉及已有素材的生成任务，必须传入参考图

## 图生图 Prompt 规则

- 使用 `--ref` 提供参考图时，prompt **只描述需要修改/新增的内容**
- 不要复述参考图里已有的外观特征（颜色、材质、风格、姿势、构图等）
- 模型会自己从参考图里提取这些信息，重复描述反而会干扰生成

## 一致性规则

- 角色外观必须与人物参考图一致
- 场景元素必须与场景参考图一致
- 同一序列的关键帧之间视觉连贯
- 画面风格与项目风格一致

## 禁止

- 禁止在需要参考图的任务中只用文字描述
- 禁止输出路径等于参考图路径（会覆盖原图）
