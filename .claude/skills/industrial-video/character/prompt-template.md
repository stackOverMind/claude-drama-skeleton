# 角色生成提示词模板

```markdown
# 人物参考-{角色名}-{特征} Prompt

## 参考图

（文生图，无参考图）

## 视觉风格

{项目风格提示词}

## 人物外貌

{性别、年龄、发型、五官、体型、特征}

## 服装

{上衣、下装、配饰、鞋}

## 人物动作

依具体场景而定（参考图中为多种表情/姿态）

## 环境描述

纯色背景（白/灰/透明）

## 光影构图

均匀柔和的光线，正面光源，无强烈阴影，全身入镜

## 完整提示词

{将以上各层合并}

## 负面 Prompt

低质量，变形，多余肢体，手指畸形，模糊，水印，文字，签名，灰暗色调，过度曝光，阴影过硬，面部崩坏，五官不对称，比例失调，背景杂乱，噪点，油画质感，西方卡通风格，3D游戏塑料感，渲染过度
```
