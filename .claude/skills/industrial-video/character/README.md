# 角色生成工作流

## 描述

为视频生成提供角色外观一致性控制。包含三视图（正面/侧面/背面）和多种表情/姿态的完整角色视觉资料，确保 AI 视频生成时角色不会变形或走样。

## 输出物

- 人物参考图大图（PNG）
- 人物提示词文件（.md）
- 可选：切割后的独立图片

## 适用场景

- 新角色首次出场，需要建立视觉基准
- 角色外观需要多角度展示
- 角色需要多种表情和姿态参考
- 作为视频生成的 `--image` 参考图输入

## 布局类型

### 布局一：左侧三视图 + 右侧 2×2 表情（配角）

```
┌──────────────────┬─────────────┬─────────────┐
│                  │   开心表情   │   愤怒表情   │
│   正面全身大图    ├─────────────┼─────────────┤
│   (1/2 宽度)      │   悲伤表情   │   战斗姿态   │
│                  │             │             │
│  包含：正面/侧面   │  (2×2 网格)  │             │
│      /背面小图    │             │             │
└──────────────────┴─────────────┴─────────────┘
```

### 布局二：左侧三视图 + 右侧 3×3 表情（主角）

```
┌──────────────────┬─────────────┬─────────────┬─────────────┐
│                  │   开心表情   │   愤怒表情   │   悲伤表情   │
│   正面全身大图    ├─────────────┼─────────────┼─────────────┤
│   (1/3 宽度)      │   惊讶表情   │   冷漠表情   │   害羞表情   │
│                  ├─────────────┼─────────────┼─────────────┤
│  包含：正面/侧面   │   战斗姿态   │   奔跑姿态   │   受伤表情   │
│      /背面小图    │             │             │             │
│                  │  (3×3 网格)  │             │             │
└──────────────────┴─────────────┴─────────────┴─────────────┘
```

## 生成流程

### Step 1 — 确认角色描述

检查项目内是否已有 `描述.md`。如果没有，先创建：

```markdown
# 角色名

## 外貌
- 性别：
- 年龄：
- 发型：
- 五官：
- 体型：
- 特征：

## 服装
- 上衣：
- 下装：
- 配饰：
- 鞋：

## 性格/气质

## 常用表情
```

### Step 2 — 选择布局

| 角色类型 | 推荐布局 | 右侧网格 |
|---------|---------|---------|
| 主角 | 左侧 1/3 + 右侧 3×3 | 9 种表情/姿态 |
| 配角 | 左侧 1/2 + 右侧 2×2 | 4 种表情/姿态 |

### Step 3 — 编写提示词文件

按 [prompt-template.md](prompt-template.md) 模板编写。

### Step 4 — 生成图片

```bash
npx -y bun .claude/skills/content-gen/scripts/image/main.ts \
  --promptfiles 人物参考-{角色名}-{特征}.md \
  --image 项目-xxx/2-素材/角色/{角色名}/人物参考-{角色名}-{特征}.png \
  --ar 1:1 --quality 2k
```

### Step 5 — 切割（可选）

```bash
npx -y bun .claude/skills/content-gen/scripts/image/grid.ts panels \
  人物参考-{角色名}-{特征}.png ./拆分 \
  --panels '[{"name":"正面","left":0,"top":0,"width":0.33,"height":0.7},{"name":"侧面","left":0,"top":0.7,"width":0.165,"height":0.3},{"name":"背面","left":0.165,"top":0.7,"width":0.165,"height":0.3},{"name":"开心","left":0.33,"top":0,"width":0.223,"height":0.33},{"name":"愤怒","left":0.553,"top":0,"width":0.223,"height":0.33},{"name":"悲伤","left":0.776,"top":0,"width":0.224,"height":0.33},{"name":"惊讶","left":0.33,"top":0.33,"width":0.223,"height":0.33},{"name":"冷漠","left":0.553,"top":0.33,"width":0.223,"height":0.33},{"name":"害羞","left":0.776,"top":0.33,"width":0.224,"height":0.33},{"name":"战斗","left":0.33,"top":0.66,"width":0.223,"height":0.34},{"name":"奔跑","left":0.553,"top":0.66,"width":0.223,"height":0.34},{"name":"受伤","left":0.776,"top":0.66,"width":0.224,"height":0.34}]'
```

## 文件存放

```
项目-漫剧-xxx/3-素材/角色/<角色名>/
├── 参考图.png           ← 人物参考图大图（必须）
├── 描述.md              ← 结构化文字描述（必须）
└── 拆分/                ← 切割后的独立图片（可选）
    ├── 正面.png
    ├── 侧面.png
    ├── 背面.png
    ├── 开心.png
    └── ...
```

## 质量检查清单

- [ ] 左侧三视图（正面/侧面/背面）完整可见
- [ ] 右侧表情/姿态网格清晰可辨
- [ ] 所有面板中角色服装一致
- [ ] 仅表情和姿态变化，外貌不变
- [ ] 背景统一为纯色（白/灰/透明）
- [ ] 画面风格与项目风格一致
