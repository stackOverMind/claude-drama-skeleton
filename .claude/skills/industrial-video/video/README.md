# 视频生成工作流

## 描述

基于关键帧或故事板生成最终视频。支持 Seedance 2.0 等视频生成模型。

## 输出物

- 视频文件（MP4）
- 视频提示词文件（.md）

## 适用场景

- 关键帧已确认，需要生成动态视频
- 故事板已确认，需要生成完整镜头
- 需要基于参考图生成特定动作的视频

## 生成流程

### Step 1 — 确认视频参数

| 参数 | 必问 | 选项 |
|------|------|------|
| 时长 | ✅ | 4-8秒 / 9-12秒 / 13-15秒 / 超长（分段） |
| 比例 | ✅ | 16:9 / 9:16 / 1:1 |
| 参考素材 | ✅ | 关键帧 / 故事板 / 角色+场景参考 |
| 风格 | 可选 | 情绪氛围、镜头风格 |

### Step 2 — 编写提示词文件

按 [prompt-template.md](prompt-template.md) 或 [seedance.md](../seedance.md) 编写。

### Step 3 — 生成视频

```bash
# 使用 content-gen skill 的视频生成功能
# 具体命令见 generation.md
```

## 文件存放

```
项目-xxx/4-视频/
├── 视频-{镜头名}-{版本}.mp4
└── 视频-{镜头名}-{版本}.md
```

## 质量检查清单

- [ ] 视频时长符合要求
- [ ] 角色外观与参考图一致
- [ ] 场景风格与参考图一致
- [ ] 动作连贯自然
- [ ] 画面风格与项目风格一致
