# 视频生成规则

## 命名规范

- 视频文件：`视频-{镜头名}-{版本}.mp4`
- 提示词文件：`视频-{镜头名}-{版本}.md`
- 存放目录：`<项目名>/4-视频/`

## 参考图规则

- 必须传入参考图（关键帧、故事板、或角色+场景参考）
- 参考图描述必须按 `--ref` 参数顺序，使用"图片1"、"图片2"等
- 多参考类型时，先列所有参考图，再列参考视频/音频

## Seedance 特定规则

- 使用 `@图片1`、`@视频1`、`@音频1` 等官方命名
- 必须在提示词中明确说明每个素材的用途
- 不要上传写实真人脸部素材
- 视频延长时，生成时长应为"新增部分"的时长

## 时长规则

| 总时长 | 推荐分段 |
|--------|----------|
| ≤15秒 | 单段生成 |
| 16-30秒 | 2段（首段15秒 + 延长段） |
| 31-45秒 | 3段 |
| 46-60秒 | 4段 |
| >60秒 | 拆分为独立场景分别生成 |

## 提示词规则

- 所有提示词必须使用中文
- 描述要具体且有画面感
- 镜头语言和动作描述要有时间顺序
- 台词/对白用引号包裹，并标注角色和情绪
- 音效描述单独成行
