# 场景生成工作流

## 描述

为视频生成提供场景视觉一致性控制。包含同一地点在不同时间、天气、景别下的完整视觉资料，确保 AI 视频生成时场景不会变形或风格漂移。

## 输出物

- 场景参考图大图（PNG）
- 场景提示词文件（.md）
- 可选：切割后的独立图片

## 适用场景

- 新场景首次出现，需要建立视觉基准
- 场景需要在不同时间/天气下保持一致
- 场景包含重要细节需要多角度展示
- 作为视频生成的 `--image` 参考图输入

## 布局类型

### 经典九宫格（3×3）— 主场景

```
┌─────────────┬─────────────┬─────────────┐
│   全景远景   │   中景构图   │   近景细节   │  ← 第1行：景别
├─────────────┼─────────────┼─────────────┤
│   白天光照   │   黄昏光照   │   夜晚光照   │  ← 第2行：时间/光照
├─────────────┼─────────────┼─────────────┤
│   晴朗天气   │   阴天氛围   │   雨天氛围   │  ← 第3行：天气
└─────────────┴─────────────┴─────────────┘
```

### 简化四宫格（2×2）— 次要场景

```
┌─────────────┬─────────────┐
│   白天全景   │   夜晚全景   │
├─────────────┼─────────────┤
│   中景细节   │   近景特写   │
└─────────────┴─────────────┘
```

## 生成流程

### Step 1 — 确认场景描述

创建或检查场景描述：

```markdown
# 场景名

## 地点
- 类型：（室内/室外）
- 规模：（房间/建筑/城市/自然）
- 地理位置：（如有特殊要求）

## 环境元素
- 建筑/地形：
- 植被：
- 天气/气候：
- 时间：

## 氛围
- 色调：
- 光线：
- 情绪：

## 关键细节
- 标志性元素：
- 道具：
```

### Step 2 — 选择布局

| 场景类型 | 推荐布局 | 理由 |
|---------|---------|------|
| 主场景 | 3×3 九宫格 | 覆盖时间、天气、景别变化 |
| 次要场景 | 2×2 四宫格 | 够用，节省生成时间 |

### Step 3 — 编写提示词文件

按 [prompt-template.md](prompt-template.md) 模板编写。

### Step 4 — 生成图片

```bash
npx -y bun .claude/skills/content-gen/scripts/image/main.ts \
  --promptfiles 场景参考-{场景名}-{视角}.md \
  --image 项目-xxx/2-素材/场景/{场景名}/场景参考-{场景名}-{视角}.png \
  --ar 1:1 --quality 2k
```

### Step 5 — 切割（可选）

```bash
# 九宫格切割
npx -y bun .claude/skills/content-gen/scripts/image/grid.ts split \
  场景参考-{场景名}-{视角}.png ./拆分 --grid 3x3 --prefix <场景名>

# 四宫格切割
npx -y bun .claude/skills/content-gen/scripts/image/grid.ts split \
  场景参考-{场景名}-{视角}.png ./拆分 --grid 2x2 --prefix <场景名>
```

## 文件存放

**必须按项目实际目录结构存放，不要机械套用模板路径。**

生成前先检查项目内已有目录：
- 如果已有 `2-素材/场景/`，就放在 `2-素材/场景/<场景名>/`
- 如果已有 `3-素材/场景/`，就放在 `3-素材/场景/<场景名>/`
- 如果两者都不存在，按项目类型标准创建：
  - 单集视频 → `2-素材/场景/<场景名>/`
  - AI 漫剧 → `3-素材/场景/<场景名>/`

```
2-素材/场景/<场景名>/
├── 参考图.png           ← 九宫格/四宫格大图（必须）
├── 描述.md              ← 结构化文字描述（必须）
└── 拆分/                ← 切割后的独立图片（可选）
    ├── <场景名>_cell_1_1.png
    ├── <场景名>_cell_1_2.png
    └── ...
```

## 质量检查清单

- [ ] 所有格子展示的是同一地点
- [ ] 建筑/环境元素保持一致
- [ ] 仅光照、天气、景别变化
- [ ] 画面风格与项目风格一致
- [ ] 标志性细节在所有格子中可识别
