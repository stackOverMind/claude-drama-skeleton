# 场景生成提示词模板

```markdown
# 场景参考-{场景名}-{视角} Prompt

## 参考图

（文生图，无参考图）

## 视觉风格

{项目风格提示词}

## 人物外貌

依具体场景而定（场景图中无人物或人物很小）

## 环境描述

{场景类型、地点、建筑/地形、植被、天气、时间}

## 人物动作

无

## 光影构图

{光源方向、时间、天气对应的光照效果}

## 完整提示词

{将以上各层合并}

## 负面 Prompt

低质量，变形，模糊，水印，文字，签名，灰暗色调，过度曝光，阴影过硬，比例失调，噪点，油画质感，西方卡通风格，3D游戏塑料感，渲染过度，人物特写，多余人物
```

## 示例

```markdown
# 场景参考-日本街道-白天 Prompt

## 参考图

（文生图，无参考图）

## 视觉风格

新海诚风格日系动画，三维渲染转二维动画质感，数字绘画，赛璐璐上色结合柔和渐变，电影级光影，高品质动画电影画面，高饱和度色彩，色彩艳丽饱满，蓝橙对比强烈

## 人物外貌

依具体场景而定

## 环境描述

日本城市街道，晴朗夏日，湛蓝天空，现代建筑，电线杆，交通标志牌，绿树，街道干净整洁，有都市生活气息

## 人物动作

无

## 光影构图

强烈的自然日光，顶部逆光，明亮的高光，柔和的阴影，蓝天白云

## 完整提示词

新海诚风格日系动画，三维渲染转二维动画质感，数字绘画，赛璐璐上色结合柔和渐变，电影级光影，高品质动画电影画面，高饱和度色彩，色彩艳丽饱满，蓝橙对比强烈，日本城市街道，晴朗夏日，湛蓝天空，现代建筑，电线杆，交通标志牌，绿树，街道干净整洁，有都市生活气息，强烈的自然日光，顶部逆光，明亮的高光，柔和的阴影，蓝天白云

## 负面 Prompt

低质量，变形，模糊，水印，文字，签名，灰暗色调，过度曝光，阴影过硬，比例失调，噪点，油画质感，西方卡通风格，3D游戏塑料感，渲染过度，人物特写，多余人物
```
