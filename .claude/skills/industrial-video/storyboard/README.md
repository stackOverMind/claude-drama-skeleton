# 故事板生成工作流

## 描述

基于关键帧或参考图生成故事板，用于视频生成或导演参考。故事板可以是单张组合图，也可以是基于关键帧序列生成的视频。

## 输出物

- 故事板图片（PNG）或视频（MP4）
- 故事板提示词文件（.md）

## 适用场景

- 需要将多个关键帧组合为一张导演参考图
- 基于关键帧序列生成连贯视频
- 预览镜头之间的过渡效果

## 两种模式

### 模式一：故事板生成单图

将多个关键帧或镜头组合为一张故事板大图。

```bash
npx -y bun .claude/skills/content-gen/scripts/image/main.ts \
  --promptfiles 故事板-{镜头名}.md \
  --image 项目-xxx/3-关键帧/故事板-{镜头名}.png \
  --ref 关键帧1.png 关键帧2.png ... \
  --ar 16:9 --quality 2k
```

### 模式二：故事板生成视频

基于故事板图或关键帧序列生成视频。

```bash
# 先生成故事板图，再生成视频
# 或直接用关键帧序列生成视频
```

## 生成流程

### Step 1 — 确认故事板内容

```markdown
# 故事板

## 镜头序列
1. [画面描述]
2. [画面描述]
3. ...

## 参考图
- 关键帧1：[路径]
- 关键帧2：[路径]
- ...
```

### Step 2 — 编写提示词文件

按 [prompt-template.md](prompt-template.md) 模板编写。

### Step 3 — 生成

根据模式选择生成图片或视频。

## 文件存放

```
项目-xxx/3-关键帧/
├── 故事板-{镜头名}.png
└── 故事板-{镜头名}.md
```

## 质量检查清单

- [ ] 镜头序列逻辑清晰
- [ ] 角色外观与参考图一致
- [ ] 场景风格与参考图一致
- [ ] 画面风格与项目风格一致
