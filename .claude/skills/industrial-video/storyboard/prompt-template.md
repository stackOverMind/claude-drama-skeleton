# 故事板生成提示词模板

## 模式一：故事板生成单图

```markdown
# 故事板-{镜头名} Prompt

## 参考图

- **图片1**：关键帧1（{描述}） `{关键帧1路径}`
- **图片2**：关键帧2（{描述}） `{关键帧2路径}`
- ...

## 视觉风格

{项目风格提示词}

## 人物外貌

{从角色参考图提取的外貌描述}

## 环境描述

{从场景参考图提取的环境描述}

## 镜头设计

每个镜头必须包含以下字段：

### 镜头{N}：{镜头名}（{时长}秒）

**景别**：{特写/近景/中景/全景/远景，必须明确指定其一}
**机位**：{固定/运动，相机高度、角度、与主体的距离}
**构图**：{主体在画面中的位置、占比。只描述画面中可见的元素，不描述画面外发生的内容}
**焦点**：{焦点位置，景深效果，虚实关系}
**动作**：
- {0.0-X秒：只描述画面中可见元素的动作，精确到秒}
- {X-Y秒：只描述画面中可见元素的动作}
**光影**：{光源方向、强度、色温，阴影位置，高光区域}

## 完整提示词

{将以上各层合并。只描述镜头内容和顺序，不指定排版格式（如行列数、格子数量、排列方向等）}

## 负面 Prompt

低质量，变形，多余肢体，手指畸形，模糊，水印，文字，签名，灰暗色调，过度曝光，阴影过硬，面部崩坏，五官不对称，比例失调，背景杂乱，噪点，油画质感，西方卡通风格，3D游戏塑料感，渲染过度
```
