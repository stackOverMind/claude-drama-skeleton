# 参考图规则

## 参考图描述规范

所有涉及参考图的生成任务，prompt 中必须**精确描述每张参考图的内容**。描述方式取决于参考图的类型和数量。

### 单张九宫格参考图

当使用单张九宫格作为参考时，prompt 中必须**逐格描述**九宫格内容：

```
参考图是一张3x3九宫格，从上往下、从左往右依次是：
图1：角色正面站姿，黑色长发，白色汉服
图2：角色侧面站姿，同一服装
图3：角色背面站姿，长发及腰
图4：角色开心表情，微笑
图5：角色愤怒表情，皱眉
图6：角色悲伤表情，低头
图7：角色惊讶表情，睁大眼
图8：角色冷漠表情，面无表情
图9：角色战斗姿态，拔剑

请生成视频：角色从冷漠表情转变为拔剑战斗姿态...
```

**规则**：
- 必须明确说「从上往下、从左往右依次是」
- 每张图用「图N：」开头编号
- 描述必须让模型知道哪张图包含什么信息，便于提取

### 多张独立参考图

当使用多张独立图片时，prompt 中必须**逐张描述**：

```
参考图共2张：
图1：角色正面全身照，黑色长发，白色汉服，微笑表情
图2：古风庭院场景，樱花树，石板路，午后阳光

请生成视频：图1中的角色站在图2的场景中，微风拂动发丝...
```

**规则**：
- 必须明确说「参考图共N张」
- 每张图用「图N：」开头编号
- 描述中引用时用「图1中的角色」「图2的场景」指代

### 起始帧 + 结束帧

当需要控制动作起止时，使用两张关键帧：

```
参考图共2张：
图1（起始帧）：角色站立在庭院中，双手自然下垂，表情平静
图2（结束帧）：同一角色拔剑出鞘，身体前倾，表情凌厉

请生成视频：从图1姿态平滑过渡到图2姿态，动作连贯自然...
```

**规则**：
- 必须标注「起始帧」和「结束帧」
- 两张图的角色必须完全一致（服装、外貌、场景）
- 仅姿态/动作不同

### 参考视频（Video-to-Video）

当需要保留动作/镜头/节奏，只修改内容时：

```
参考视频：一只柴犬挥舞刀盾的动作视频

请生成视频：完整保留参考视频中的所有动作、节奏、镜头运动、光影效果——
仅将柴犬替换为橘猫，橘猫保持与柴犬完全一致的动作轨迹...
```

**规则**：
- 必须明确说「完整保留...」然后才说修改点
- 修改点要具体，不要笼统说「换成别的」

### 参考音频（Audio-to-Video）

当需要根据音频生成对口型或舞蹈时：

```
参考音频：一段中文女声朗诵，语速中等，情感忧伤
参考图共1张：角色正面照，年轻女性，黑色长发，白色汉服

请生成视频：图1中的角色根据参考音频做口型同步，表情忧伤，
头部微微低垂，眼神看向下方...
```

**规则**：
- 音频和图像参考都要描述清楚
- 明确说「根据参考音频做...」

### 组合参考

可以同时使用多种参考类型：

```
参考图共2张：
图1（角色九宫格）：从上往下、从左往右依次是——正面/侧面/背面/开心/愤怒/悲伤/惊讶/冷漠/战斗
图2（场景）：古风庭院，樱花树，午后阳光

参考音频：忧伤的古琴曲

请生成视频：图1中正面站姿的角色站在图2场景中，
根据参考音频的情绪，表情从冷漠渐变为忧伤...
```

**多参考类型的描述优先级**：
1. 先列所有参考图，逐张/逐格描述
2. 再列参考视频/音频，描述内容
3. 最后写生成要求，引用「图N」「参考视频」「参考音频」指代

## 参考图强制规则

**任何涉及已有素材的生成任务，必须传入参考图。**

| 场景 | 必须传入的参考 | 说明 |
|------|---------------|------|
| 基于场景生成角色动作 | 场景参考图 | 确保角色与场景光影/色调一致 |
| 基于角色生成新姿势 | 角色参考图 | 确保角色外貌/服装一致 |
| 角色在场景中做动作 | 角色参考图 + 场景参考图 | 两者都必须 |
| 修改已有图片风格 | 原图作为参考 | 图生图模式 |

**错误示例（禁止）**：
- ❌ 用户说「参考场景第五格生成女孩滑板」，但只传文字 prompt 不传场景图
- ❌ 用户说「让Yi做这个动作」，但不传Yi的参考图
- ❌ 用文字描述「海边悬崖公路正午场景」代替传入已生成的场景参考图

**正确做法**：
- ✅ 先确认已有哪些参考图（场景九宫格、角色参考图）
- ✅ 如果不确定用哪张参考图，**询问用户，不要自作主张**
- ✅ 用 `--ref` 传入所有相关参考图
- ✅ prompt 中按「图1是...图2是...」规则描述每张参考图

## 参考图编号规则

在 prompt 文件中指代参考图时，**必须按 `--ref` 参数中的提交顺序**，使用"图片1"、"图片2"等，**绝不能出现文件名**。

| `--ref` 参数顺序 | prompt 中指代 |
|-----------------|--------------|
| `--ref 角色图.png` | 图片1 |
| `--ref 角色图.png 场景图.png` | 图片1、图片2 |
| `--ref 角色图.png 场景图.png 首帧.png` | 图片1、图片2、图片3 |
