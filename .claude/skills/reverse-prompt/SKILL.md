# 反推提示词

根据图片反推出用于 AI 图像生成的高质量提示词。

## 用法

```
/reverse-prompt <图片路径>
```

## 输出格式

分析图片后，按以下结构输出：

```markdown
## 画面分析

- 主体：...
- 场景/背景：...
- 风格：...
- 光影：...
- 构图：...
- 色彩：...
- 质感/细节：...

## 生成提示词

### 视觉风格
<画面整体的艺术风格、渲染方式、绘画技法>

### 人物外貌
<人物的长相、发型、五官、表情、穿着、配饰>

### 环境描述
<场景、背景、地点、时间、氛围>

### 人物动作
<姿态、动作、肢体语言、与镜头的互动关系>

### 光影构图
<光源、明暗、镜头角度、景别、画面布局>

### 完整提示词
<将以上各层合并为一段完整的中文提示词，按重要性排序，用逗号分隔>

## 负面提示词

<一段完整的中文负面提示词，排除画面中不希望出现的元素和常见问题>

## 建议参数

- 比例：...
- 风格化程度：...
```

## 规则

- 提示词必须为纯中文
- 分层输出各维度描述，方便用户单独取用或调整
- 最后提供一段合并的完整提示词，按重要性排序，用逗号分隔
- 按重要性排序：视觉风格 > 人物外貌 > 人物动作 > 环境描述 > 光影构图 > 色彩 > 细节
- 负面提示词覆盖：低质量、变形、多余肢体、模糊、水印等常见问题
- 如果图片包含特定艺术风格（如吉卜力、赛博朋克、水墨画等），务必在视觉风格层准确描述
- 如果图片有特定镜头效果（如景深、广角、特写等），在光影构图层描述
